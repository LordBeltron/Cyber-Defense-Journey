# Lessons Learned

## Active Directory

Identity should be treated as the enterprise control plane.

Organizational Units should reflect administrative boundaries rather than organizational charts.

Group Policy should be applied using least privilege and clear organizational structure.

---

## Group Policy

Test policies before broad deployment.

Always document GPO changes.

Backup policies before modification.

---

## DNS

Forward lookup zones resolve hostnames to IP addresses.

Reverse lookup zones resolve IP addresses to hostnames.

Reverse lookup zones improve troubleshooting and enterprise interoperability.

DNS is a foundational dependency for Active Directory.

Validate DNS after every configuration change.

---

## Documentation

Documentation is part of the implementation.

Engineering artifacts should be completed immediately after each lab while implementation details remain fresh.
